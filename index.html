Quick Start

Get productive in 3 commands.

1

Initialize your project

Run fwts init in your repo to create a config file.

2

Start a feature

Run fwts start TICKET-123 or fwts start my-feature-branch.

3

Open the dashboard

Run fwts to see all your worktrees and manage them interactively.

4

Set up Claude Code

Copy this prompt and paste it into Claude:

Set my Claude statusline to: $(fwts statusline)

The Workflow

From ticket to shipped, seamlessly.

1

Start a feature

fwts start SUP-123 ‚Äî Creates worktree, starts Docker, opens tmux, runs setup hooks.

2

Work in isolation

Each feature has its own directory, tmux session, and Docker services. No conflicts.

3

Switch context

fwts focus hotfix-branch ‚Äî Shared resources switch over automatically.

4

Clean up

fwts cleanup ‚Äî Worktree deleted, tmux killed, Docker stopped, branch removed. One command.
